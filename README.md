# Implementing and training GPT-2 124M from Scratch

This repositry contains a re-implementation of the GPT-2 (124M) model from OpenAI introduced in 2019.

This project is inspired by Andrej Karpathy’s [nanoGPT](https://github.com/karpathy/build-nanogpt) and his accompanying [YouTube video walkthrough](https://www.youtube.com/watch?v=l8pRSuU81PU).
